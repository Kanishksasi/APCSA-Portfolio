# APCSA-Portfolio

A collection of Java projects and exercises completed throughout AP Computer Science A (PLTW curriculum). This repository showcases progression in object-oriented programming, logic design, and algorithmic thinking.

## Projects

**Code Challenges** — Independent coding challenges focused on problem-solving and Java fundamentals, completed outside of standard coursework.

**PLTW 1.2 — Objects & Methods** — Introductory OOP concepts including class creation, string manipulation, and Java Math methods. Features a MadLibs project built with string operations.

**PLTW 1.3 — Conditionals & Boolean Logic** — Programs using if/else statements, compound boolean expressions, and object equality. Includes an interactive Guess Checker game with multiple difficulty variations.

**PLTW 2.5 — Arrays & ArrayLists** — Multi-class Java programs demonstrating encapsulation and data structures. Features a Wheel of Fortune-style PhraseSolver game (Board, Player, PhraseSolver, Runner classes) and a Media Library system managing Books, Movies, and Songs.

---

**Languages:** Java  
**Topics:** OOP, Encapsulation, Conditionals, Loops, ArrayLists, Multi-class Design
